# IT Opex Budget Management

Responsive web app to capture and compare IT Opex data across 22 locations.

## Features
- Add, edit, and delete expense lines.
- Compare `Last Year Expense` vs `Current Year Budget`.
- Category and location insights with variance and share.
- Coding dropdown populated with the provided `ITOPEX...` values.
- Filter/search by location, category, and free text.
- Local persistence (`localStorage`).
- JSON export/import for backup and transfer.

## Run
1. Open [`index.html`](./index.html) in a browser.
2. Start entering budget records in **Capture Expense**.

No build step is required.
